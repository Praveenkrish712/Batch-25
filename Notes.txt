HTML5 - Hyper text markup language (<head> content </head>)

Self closing tags - hr, Meta, br, img

===> It is not a programming language
===> it is markup language
===> Every website can start with HTML
===> To create a Web page

1) Heading - h1 to h6

2) Paragraph - p

3) Horizontal - hr

4) Break Tag - br

5) Text Formatting - U, i, b or strong, sub, sup and Mark

6) Anchor Tag (Linking) - a

7) Image tag - img

8) List - Orderd list (ol) and unordered list(ul)

9) tables - tr, th and td
