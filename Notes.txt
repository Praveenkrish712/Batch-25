CSS - Cascading Style Sheet
---> it is used to Style and design a web page
---> CSS works together with HTML
---> Change the text, color and background color
---> to set the image and design a layout
---> Adding a Animation and Transition

1) Types Of CSS
i - Inline CSS
ii - Internal CSS
iii - External CSS
Inline > Internal > External

2) CSS Syntax
Selector{
    property:value;
}

3) Types of Selector
i - Elements Selector - h1, h2, p 
ii - Class Selector - (.) - Multiple - we can use the Same Class name in no of times
iii - Id Selector - (#) - Unique - we can use the same id name in only one time
iv - Universal Selector - (*) - Style can apply the entire site
v - Group Selector - Class or Id name contains different tags, in that different tags we want to change the same Style means we can go with Group Selector

7) Position 
i - Static - Default
ii - Relative - Moveing one place to another place
iii - Absolute - Absolute can search the non-Static parent, once reach the non-Static it will act as a same Relative, otherwise it will search one more...... non-Static parent, there is no non-Static parent means it will directy go to the root(body) of the web site/page
iv - Fixed - to be fixed to the place
v - Sticky - Mostly using header part and using top:0 once reach the top:0 it will act as a same fixed

8) Display - block, inline, inline-block, flex, grid and none

         Flex                                 grid
   One Dimensional layout               Two Dimensional layout
   Row OR Column                         Row and Column
   Best for small components          Best for Full page layout
   Ex:Navbar, cards                    Ex: section, dashboard

====> Flex
i) Using Display:flex to the parent it will works into child,
change Column into Row 
ii) Flex-direction - Row, row-reverse, Column, Column-reverse
iii) justify-content (X) - Start, center, end, space-Around, space-evenly, space-between
iv) Align-items (Y) (set the height) - Start, center, end
v) gap - to be set the value (px,cm,m,%,rem,vh,vw)
vi) Flex-wrap - nowrap, wrap, wrap-reverse
vii) Align-content (Y) (set the height) (flex-wrap is enable) - Start, center, end, space-Around, space-evenly, space-between
viii) flex on item - flex:1,2,3....
iX) Order - We can Change the Visual order of the property
